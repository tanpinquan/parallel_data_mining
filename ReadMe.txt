Four algorithms are implemented in sequential and parallel. 
Run the specific file listed below to execute the implementation

---------
Libraries used:
numpy
pandas
matplotlib
multiprocessing
time
csv
collections

----------
APriori
----------
apriori_test_seq: Run to perform sequential Apriori
apriori_test_par: Run to perform parallel Apriori
apriori_seq: functions for sequential Apriori
apriori_par: functions for parallel Apriori

----------
ECLAT
----------
eclat_test_seq: Run to perform sequential ECLAT
eclat_test_par: Run to perform parallel ECLAT
eclat_seq: functions for sequential ECLAT
eclat_par: functions for parallel ECLAT

----------
K Means
----------
kmeans_test_seq: Run to perform sequential K Means
kmeans_test_par: Run to perform parallel K Means
kmeans_test_seq_fixed: Run to perform sequential K Means for 10 iterations
kmeans_test_par_fixed: Run to perform parallel K Means for 10 iterations
kmeans_seq: functions for sequential K Means
kmeans_par: functions for parallel K Means

----------
Hierarchical
----------
hiearch_test_seq: Run to perform sequential Hierarchical Clustering
hiearch_test_par: Run to perform parallel Hierarchical Clustering
hiearch_test_par_2: Run to perform modified parallel Hierarchical Clustering
hiearch_seq: functions for sequential Hierarchical Clustering
hiearch_par: functions for parallel Hierarchical Clustering
